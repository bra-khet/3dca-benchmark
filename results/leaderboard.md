## How to add a row

1. Archive under `runs/<YYYY-MM-DD>_<model-slug>/`.
2. Score using `meta.json` fields.
3. Append a row here with a short summary (one line).

## Conventions

- **Opens / Sim / Ray / UI:** `Y` / `N` / `~` (partial)
- **Visual / Code / FPS:** integers 1–5
- **Path:** relative link, e.g. `../runs/2026-07-09_example/output.html`
